## [704. 二分查找](https://leetcode.cn/problems/binary-search/description/)
给定一个 `n` 个元素有序的（升序）整型数组 `nums` 和一个目标值 `target`  ，写一个函数搜索 `nums` 中的 `target`，如果 `target` 存在返回下标，否则返回 `-1`。

你必须编写一个具有 `O(log n)` 时间复杂度的算法。

  
**示例 1:**

**输入:** `nums` = [-1,0,3,5,9,12], `target` = 9
**输出:** 4
**解释:** 9 出现在 `nums` 中并且下标为 4

**示例 2:**

**输入:** `nums` = [-1,0,3,5,9,12], `target` = 2
**输出:** -1
**解释:** 2 不存在 `nums` 中因此返回 -1

**提示：**

1. 你可以假设 `nums` 中的所有元素是不重复的。
2. `n` 将在 `[1, 10000]`之间。
3. `nums` 的每个元素都将在 `[-9999, 9999]`之间。

我的解法：
1.最直观，用 ".index" 来得到列表索引，这种方式最好O(N)，最坏O(n^2)
```python
class Solution(object):
    def search(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: int
        """
        a=-1
        for i in nums:
            if i == target:
                a=nums.index(i)
            else:
                continue
        return a
```
2.==有序数组+无重复==-->直接==二分查找==，O(log n)
```python
class Solution(object):
    def search(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: int
        """
        left , right = 0 , len(nums)-1
        while(left <= right):
            mid = (left + right )//2
            if nums[mid] < target :
                left = mid + 1
            elif nums[mid] > target :
                right = mid - 1
            else:
                return mid
        return -1
```

## [35. 搜索插入位置](https://leetcode.cn/problems/search-insert-position/)
给定一个排序数组和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。

请必须使用时间复杂度为 `O(log n)` 的算法。

**示例 1:**

**输入:** nums = [1,3,5,6], target = 5
**输出:** 2

**示例 2:**

**输入:** nums = [1,3,5,6], target = 2
**输出:** 1

**示例 3:**

**输入:** nums = [1,3,5,6], target = 7
**输出:** 4

**提示:**

- `1 <= nums.length <= 104`
- `-104 <= nums[i] <= 104`
- `nums` 为 **无重复元素** 的 **升序** 排列数组
- `-104 <= target <= 104`

我的解法
- 为什么直接最后return left呢，因为是要插入列表中，
  假如是在右边界，那就是nums[mid]< target ,是left最后被更新的，此时left不可能大于right，只能等于right （因为mid=(left+right)//2，是向下取整）
  假如是左边界，那就是nums[mid]> target，是right最后被更新的，有可能等于left也有可能小于left，所以只要返回left就是一定是插入位置
```python
class Solution(object):
    def searchInsert(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: int
        """
        left , right = 0,len(nums)-1
        while(left <=right):
            mid=(left+right)//2
            if nums[mid]<target:
                left = mid +1
            elif nums[mid]>target:
                right = mid -1
            else:
                return mid
        return left
```

如果数组中不存在目标值 `target`，返回 `[-1, -1]`。

你必须设计并实现时间复杂度为 `O(log n)` 的算法解决此问题。

**示例 1：**

**输入：** nums = [`5,7,7,8,8,10]`, target = 8
**输出：**[3,4]

**示例 2：**

**输入：** nums = [`5,7,7,8,8,10]`, target = 6
**输出：**[-1,-1]

**示例 3：**

**输入：** nums = [], target = 0
**输出：**[-1,-1]

**提示：**

- `0 <= nums.length <= 105`
- `-109 <= nums[i] <= 109`
- `nums` 是一个非递减数组
- `-109 <= target <= 109`

我的解法：
在findright的时候，`python if mid==len(nums)-1 or nums[mid+1]>target :` 这个代码因为or运算逻辑短路，当已经是nums中最后一个元素的时候，第一个成立，就不会进行第二个判断，直接就不会抛出IndexError
就是确定两个边界，就可以了，二分变成查找最左边以及最右边的元素就可以了

```python
class Solution(object):

    def findleft(self, nums, target):

        left , right = 0,len(nums)-1

        while(left<=right):

            mid=(left+right)//2

            if nums[mid]==target:

                if mid==0 or nums[mid-1]<target :

                    return mid

                else:

                    right = mid-1

            elif nums[mid]<target:

                left = mid+1

            else:

                right = mid-1

        return -1

  

    def findright(self, nums, target):

        left , right = 0,len(nums)-1

        while(left<=right):

            mid=(left+right)//2

            if nums[mid]==target:

                if mid==len(nums)-1 or nums[mid+1]>target :

                    return mid

                else:

                    left = mid+1

            elif nums[mid]<target:

                left = mid+1

            else:

                right = mid-1

        return -1

    def searchRange(self, nums, target):

        """

        :type nums: List[int]

        :type target: int

        :rtype: List[int]

        """

        return [self.findleft(nums,target),self.findright(nums,target)]
        
```

## [69. x 的平方根](https://leetcode.cn/problems/sqrtx/) 
给你一个非负整数 `x` ，计算并返回 `x` 的 **算术平方根** 。

由于返回类型是整数，结果只保留 **整数部分** ，小数部分将被 **舍去 。**

**注意：**不允许使用任何内置指数函数和算符，例如 `pow(x, 0.5)` 或者 `x ** 0.5` 。

**示例 1：**

**输入：** x = 4
**输出：** 2

**示例 2：**

**输入：** x = 8
**输出：** 2
**解释：** 8 的算术平方根是 2.82842..., 由于返回类型是整数，小数部分将被舍去。

**提示：**

- `0 <= x <= 2^31 - 1`
我的解法：
1.这样有很大的问题，这样是O（N），遇到比较长的会报错，循环时间太长了，其次这种x//2+2的形式其实是因为在处理1的时候会报错，不得不这样+2规避，所以正确做法应该是找一个logn的做法
```python
class Solution(object):
    def mySqrt(self, x):
        """
        :type x: int
        :rtype: int
        """
        for i in range(0,(x//2)+2):
            if i*i<=x and (i+1)*(i+1)>x:
                return i
```

2.寻求一个Ologn的算法，二分搜索解决这个问题
我一开始想到的是直接两个二分，一个搜索平方小于它的数，然后去缩小左边界，同时去缩小有边界，最后应该返回左边界-1，因为最后一次更新的时候是mid+1，所以应该最后left-1
```python
class Solution(object):

    def mySqrt(self, x):
        """
        :type x: int
        :rtype: int
        """
        left,right=0,x
        while(left<=right):
            mid=(left+right)//2
            if(mid*mid<=x):
                left = mid+1
            elif(mid*mid>x):
                right = mid-1
                
        return left-1
```

给你一个正整数 `num` 。如果 `num` 是一个完全平方数，则返回 `true` ，否则返回 `false` 。

**完全平方数** 是一个可以写成某个整数的平方的整数。换句话说，它可以写成某个整数和自身的乘积。

不能使用任何内置的库函数，如  `sqrt` 。

**示例 1：**

**输入：** num = 16
**输出：** true
**解释：** 返回 true ，因为 4 * 4 = 16 且 4 是一个整数。

**示例 2：**

**输入：** num = 14
**输出：** false
**解释：** 返回 false ，因为 3.742 * 3.742 = 14 但 3.742 不是一个整数。

**提示：**

- `1 <= num <= 2^31 - 1`
我的解答:
很简单，直接二分就可以，标准板子题目
```python
class Solution(object):
    def isPerfectSquare(self, num):
        """
        :type num: int
        :rtype: bool
        """
        left,right=0,num
        while(left<=right):
            mid=(left+right)//2
            if (mid*mid<num):
                left=mid+1
            elif(mid*mid>num):
                right=mid-1
            else:
                return True
        return False
```

class Solution(object):

                slow+=1

                nums[slow]=nums[fast]

            fast+=1

**示例 2：**

**输入：**nums = [-7,-3,2,3,11]
**输出：**[4,9,9,49,121]

**提示：**

- `1 <= nums.length <= 104`
- `-104 <= nums[i] <= 104`
- `nums` 已按 **非递减顺序** 排序

**进阶：**

- 请你设计时间复杂度为 `O(n)` 的算法解决本问题
我的解答：
偷懒，直接调用库直接解决，这个时间复杂度是 O(n + nlogn)， 可以说是O(nlogn)的时间复杂度
```python
class Solution(object):

    def sortedSquares(self, nums):

        """

        :type nums: List[int]

        :rtype: List[int]

        """

        a=[i*i for i in nums ]

        a.sort()

        return a
```

- 需要设计一个时间复杂度为 `O(n)` 的算法解决本问题，考虑双指针，这样直接就两个首尾比较就可以，赋值给新的列表
```python
class Solution(object):

    def sortedSquares(self, nums):

        """

        :type nums: List[int]

        :rtype: List[int]

        """

        i,j,k=0,len(nums)-1,len(nums)-1

        a=[0]*len(nums)

        while(i<=j):

            if(nums[i]*nums[i]>nums[j]*nums[j]):

                a[k]=nums[i]*nums[i]

                i+=1

            else:

                a[k]=nums[j]*nums[j]

                j-=1

            k-=1

        return a
```
